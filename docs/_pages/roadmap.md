## 年度別の到達目標（KPI）

事業期間は令和8年度（R8）から令和10年度（R10）までの3年間です。各テーマの年度別の到達目標は以下の通りです。
（KPIの各指標には技術成熟度 TRL・事業成熟度 BRL を併記しています。）

### テーマ①：教育特化型AIモデルおよびエージェント技術の研究開発

| 年度 | 主な到達目標（KPI） |
| :-- | :-- |
| **R8** | 教育データ・学習データの収集・構築・整備。教育特化型AI評価ベンチマーク整備（ハルシネーション率、有害回答率、指導要領不適合率）。教育特化型AIプロトタイプモデルをリリースし、上記ベンチマーク指標で5%以下を実現（TRL4） |
| **R9** | 教育データ・学習データ・評価ベンチマークの収集・改善・公開。教育特化型AIモデル改修版リリースで3%以下を実現。汎用エージェント基盤の開発、評価ベンチマーク1%以下を実現（TRL5） |
| **R10** | 実証実験結果を反映し、実運用可能な最終成果物（データ・AIモデル・エージェント基盤・スクリプト）を整備・公開。GIGAスクール端末対応版リリース（TRL7） |

### テーマ②：教育改善メタエージェントおよびアナリティクス技術の研究開発

| 年度 | 主な到達目標（KPI） |
| :-- | :-- |
| **R8** | 教育改善メタエージェントのプロトタイプリリース。教育プロセスデータ収集 100,000件。教育改善ベンチマーク整備（理解度推定、教育改善提案）（TRL7 BRL5） |
| **R9** | 教育改善メタエージェント改修版リリース。教育プロセス収集 1,000,000件。教育改善ベンチマークで理解度向上5%以上・教育改善提案採用率30%以上。教員アンケートによる授業改善実現50%以上（TRL7 BRL6） |
| **R10** | 教育改善メタエージェント完成版・GIGAスクール端末対応版リリース。理解度向上10%以上・教育改善提案採用率50%以上。授業改善実現70%以上（TRL7 BRL7） |

### テーマ③：学習者支援・教員支援エージェントの開発と実証研究

| 年度 | 主な到達目標（KPI） |
| :-- | :-- |
| **R8** | 学習者支援・教員支援エージェントのプロトタイプリリース。教員支援ベンチマーク整備（授業準備・事務作業削減時間、満足度）、学習者支援ベンチマーク整備（評価テストスコア・IRTスコア 等）（TRL7 BRL5） |
| **R9** | 各エージェント改修版リリース。教員支援ベンチマークで10%以上改善、学習者支援ベンチマークでスコア差分10%以上。非導入校に対する有意差を実現（TRL7 BRL6） |
| **R10** | 各エージェント完成版・GIGAスクール端末対応版リリース。教員支援ベンチマークで削減時間10%以上・満足度80%以上を実現（TRL7 BRL7） |

## 工程表（概略）

| テーマ | R8年度 | R9年度 | R10年度 |
| :-- | :-- | :-- | :-- |
| ①モデル・エージェント技術 | 学習・評価データ構築、プロトタイプ開発 | 収集・整備、プロトタイプ改修 | リソース公開、実運用可能な成果として整備 |
| ②教育改善メタエージェント | プロトタイプ開発 | 第一次評価・システム改良 | システム完成 |
| ③支援エージェント・実証研究 | プロトタイプ開発・実証 | システム改修・実証 | 完成版開発・実証 |
| 教育AI人材育成 | 教育AIセミナー企画 | 教育AIセミナー・オンデマンド化 | 教育AIセミナー・スキル認定化 |

> 事業の年度計画は、事業の進捗および世の中のニーズ・シーズを踏まえて変動しうるものです。

## 令和8年度（初年度）の主な取り組み

- **教育データ収集及び整備**：教育関係資料の収集（自動・人手）および整備
- **学習データ構築**：学習データ設計 → アノテーション → 学習検証
- **定量評価用データ構築**：評価データ設計 → アノテーション → 既存モデル評価
- **教育特化型AIモデルプロトタイプリリース**：プロトタイプ教育特化型AIの学習
- **メタエージェント・支援エージェント**：プロトタイプ開発と改良、実証校での実証
- **教育AI人材育成**：教育AIセミナーの開催

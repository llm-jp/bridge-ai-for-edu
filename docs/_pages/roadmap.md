事業期間は令和8年度（R8）から令和10年度（R10）までの3年間です。各テーマの年度別の到達目標は以下の通りです。

### テーマ1：教育特化型AIモデルおよびエージェント技術の研究開発

| 年度 | 主な到達目標 |
| :-- | :-- |
| **R8** | 教育データ・学習データの収集・構築・整備。教育特化型AI評価ベンチマーク整備（ハルシネーション率、有害回答率、指導要領不適合率）。教育特化型AIプロトタイプモデルをリリースし、上記ベンチマーク指標で5%以下を実現 |
| **R9** | 教育データ・学習データ・評価ベンチマークの収集・改善・公開。教育特化型AIモデル改修版リリースで3%以下を実現。汎用エージェント基盤の開発、評価ベンチマーク1%以下を実現 |
| **R10** | 実証実験結果を反映し、実運用可能な最終成果物（データ・AIモデル・エージェント基盤・スクリプト）を整備・公開。GIGAスクール端末対応版リリース |

### テーマ2：教育改善メタエージェントおよびアナリティクス技術の研究開発

| 年度 | 主な到達目標 |
| :-- | :-- |
| **R8** | 教育改善メタエージェントのプロトタイプリリース。教育プロセスデータ収集 100,000件。教育改善ベンチマーク整備（理解度推定、教育改善提案） |
| **R9** | 教育改善メタエージェント改修版リリース。教育プロセス収集 1,000,000件。教育改善ベンチマークで理解度向上5%以上・教育改善提案採用率30%以上。教員アンケートによる授業改善実現50%以上 |
| **R10** | 教育改善メタエージェント完成版・GIGAスクール端末対応版リリース。理解度向上10%以上・教育改善提案採用率50%以上。授業改善実現70%以上 |

### テーマ3：学習者支援・教員支援エージェントの開発と実証研究

| 年度 | 主な到達目標 |
| :-- | :-- |
| **R8** | 学習者支援・教員支援エージェントのプロトタイプリリース。教員支援ベンチマーク整備（授業準備・事務作業削減時間、満足度）、学習者支援ベンチマーク整備（評価テストスコア・IRTスコア 等） |
| **R9** | 各エージェント改修版リリース。教員支援ベンチマークで10%以上改善、学習者支援ベンチマークでスコア差分10%以上。非導入校に対する有意差を実現 |
| **R10** | 各エージェント完成版・GIGAスクール端末対応版リリース。教員支援ベンチマークで削減時間10%以上・満足度80%以上を実現 |

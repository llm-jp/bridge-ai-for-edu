## プロジェクト概要

**BRIDGE教育AI**は、文部科学省「研究開発とSociety 5.0との橋渡しプログラム（BRIDGE）」のもとで進める、
**「教育向けAIモデルの構築と人と協調するエージェントAI教育支援エコシステムの開発」** プロジェクトです。

国立情報学研究所（NII）を主体として、京都大学、大阪教育大学が連携し、教育関連企業や実証校とともに、
日本の教育制度に適合した透明性・信頼性の高い**教育特化型AIモデル**と、それを中核とする**3種のエージェントAI**
（学習者支援・教員支援・教育改善）からなる教育支援エコシステムを開発します。

> エージェントAIとは：AIモデルを中核とし、与えられた目標に対して自律的に行動するAIのことです。

## 解決すべき社会課題

国土や資源に制約を持つ日本にとって、人材は持続的発展の基盤であり、その能力を引き出す教育、
とりわけAI活用への戦略的投資は重要です。しかし、日本の教育現場におけるAI活用は大きく立ち遅れています。

- 国際調査（TALIS2025）では、日本の中学校教員のAI活用率は**55か国中54位**にとどまる。
- 学習・教育プロセスに関するデータが体系的に蓄積・活用されておらず、エビデンスに基づく教育の高度化が進まない。
- 教育現場でAIを安心・効果的に普及させるための、透明性と安全性を備えた教育特化型AIの共通基盤が整っていない。
- 専門的ノウハウや大規模計算資源を要するため民間参入は難しく、既存AIの利活用にとどまっている。

## 私たちのアプローチ

文部科学省の「生成AIモデルの透明性・信頼性の確保に向けた研究開発拠点形成」により構築される、
日本語に強い高性能な大規模言語モデル（LLM-jp）を基盤とし、教育特化型AIモデルを構築します。
その上で、学習者支援・教員支援・教育改善を担う3種のエージェントAIからなる教育支援エコシステムを開発し、
高等学校および小中学校等における実証研究を通じて有効性・安全性を評価します。

### 3種のエージェントAI

| エージェント | 役割 |
| :-- | :-- |
| **学習者支援エージェント** | 学習者の理解度・関心・学習履歴に基づき、個別最適化された学習支援とフィードバックを提供 |
| **教員支援エージェント** | 授業計画、自動フィードバック、教材作成、問題作成・採点支援などを通じて、指導準備の効率化と指導力向上に寄与 |
| **教育改善メタエージェント** | 各エージェントから得られる教育プロセスを分析し、知見を体系的に蓄積、教育特化型AIモデル及びエージェントを半自動改善 |

詳しくは[研究概要のページ]({{ "/research/" | relative_url }})をご覧ください。

## 目指す社会実装

教育特化のAIモデルとエージェントAIの共通基盤を整備し、教育現場に導入することで、生徒への個別最適な学習支援、
教員の指導方法の改善や負担軽減を実現します。AIと教育実践が相互に学習し進化する循環型の学習支援環境を実現し、
我が国全体の教育水準の向上とAI技術基盤の強化、「世界で最もAIを開発・活用しやすい国」の実現に貢献します。
